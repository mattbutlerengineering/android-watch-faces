# Android Watch Faces

A collection of Android watch faces built using the [Watch Face Format (WFF)](https://developer.android.com/training/wearables/wff).

## Projects

- `digital-basic`: Digital watch face with battery, heart rate, date, and user-customizable color themes.
- `analog-basic`: A simple analog watch face.
- `analog-modern`: Sophisticated analog face with radial gradients, integrated complications, and smooth seconds.
